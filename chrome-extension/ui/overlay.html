How to create a Slack Workflow with Webhook:

  1. Open Slack and click your workspace name > Tools > Workflow Builder.
  2. Click Create Workflow and choose From scratch.
  3. Select Webhook as the trigger.
  4. Add a Send a message step. Select a channel and customize the message.
  5. Publish the workflow and copy the Weblink URL.

Note: You can also use traditional Slack Apps with Incoming Webhooks if preferred.
